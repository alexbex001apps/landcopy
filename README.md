## Incluye
- Landing page completa
- WhatsApp
- Redes sociales
- TikTok
- Email
- Objeciones
- Descripción de tienda
- SEO
- Botones de copiar
- Backend seguro en `/api/generate`

## Instalación en Vercel
1. Descomprime el ZIP.
2. Sube la carpeta a Vercel.
3. Ve a Settings → Environment Variables.
4. Agrega:
   OPENAI_API_KEY = tu_api_key
5. Publica el proyecto.

La API Key no está dentro del HTML.
